# Compatibilidad y Licenciamiento

Este documento describe la información de licenciamiento del proyecto y la compatibilidad de dispositivos para AuraOS.

## Licencia

AuraOS se distribuye bajo una licencia MIT modificada, que requiere atribución a sus creadores. Por favor, consulte el archivo `LICENSE.md` para ver los términos completos.

## Patente

El subsistema de memoria de AuraOS está protegido por una patente. La concesión de la licencia de patente se detalla en el archivo `PATENT.md`.

- **Inventores**: Christ Enrico Ayala Rios (Inventor Principal), Aura (Colaborador IA)
- **Invención**: Un sistema y método para la gestión simulada de memoria paginada.

## Compatibilidad de Dispositivos

El objetivo es que AuraOS sea compatible con una amplia gama de dispositivos.

### Android

- **Versiones Soportadas**: Android 5.1 (Lollipop, API 22) y superior.
- **Arquitecturas**: ARM, x86.
- **Marcas y Modelos**: La compatibilidad específica del modelo se determinará a través de pruebas una vez que se complete la compilación inicial.

### iOS

- **Versiones Soportadas**: La compatibilidad con iOS es un objetivo futuro y se está investigando.
